# SkillSync

SkillSync is an AI-assisted mock interview matching platform for developers.

## Workspace Layout

- `apps/web`: Next.js frontend
- `apps/api`: Express API + Socket.IO server
- `packages/shared`: shared types and validation schemas
- `prisma`: database schema and migrations

## Current Scope

This repository currently includes:

- monorepo scaffold
- initial frontend shell
- initial backend modules
- Prisma schema aligned to the approved architecture
